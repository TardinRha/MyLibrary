# MyLibrary

Aplicação web desenvolvida para salvar livros lidos e livros que quer ler (não lidos).

## Demonstração

https://tardinrha.github.io/MyLibrary/

## Funcionalidades

- Adicionar livro na lista
- Editar livro
- Deletar livro
- Limpar lista

## Tecnologias Utilizadas

- JavaScript
- HTML
- CSS
